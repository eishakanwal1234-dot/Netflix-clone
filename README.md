# Netflix UI Clone 🎬

A modern, responsive recreation of the Netflix landing page. This project focuses on building a pixel-perfect user interface and mastering complex CSS layouts.

## 🚀 Features
- **Fully Responsive Design:** Optimized for mobile, tablet, and desktop screens.
- **Modern Layouts:** Built using CSS Flexbox and CSS Grid for structured content.
- **Interactive UI:** Includes hover effects on movie thumbnails and a functional FAQ accordion section.
- **Clean Code:** Semantic HTML5 used for better accessibility and SEO.

## 🛠️ Tech Stack
- **HTML5:** For the structure of the webpage.
- **CSS3:** For styling, animations, and responsive design.
- **Font Awesome:** For professional-looking icons.

## 🎯 Learning Objectives
This project was a deep dive into:
1. Translating a professional design into functional code.
2. Managing complex positioning and layering (Z-index, Absolute/Relative).
3. Optimizing assets for faster web performance.
